UI Components / Accordion

The <accordion> element requires the inclusion of NG-2 Bootstrap - an external resource used to create Bootstrap-like elements in Angular 2 applications. NG-2 Bootstrap was renamed to NGX-Bootstrap. Please refer to this documentation when customizing this component.
