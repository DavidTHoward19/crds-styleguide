Video modal

To get the video modal to show display properly, add .modal-video to the wrapping modal div.

Fullscreen Modals

Full-screen modals slightly deviate from default modals in that they require an additional sizing class modifier to achieve the full-screen effect. For this modal to work, you must add a class of .modal-full to the .modal-dialog <div>. Full-screen modals also use the .alpha class in place of the .fade class to achieve a smoother animation.
