Checkboxes

Select none, one or multiple options. No option is preselected by default. Once an option is touched, a user can return to the unselected state of the component.

The button groups featured in this section have radio button behaviour associated with them. This is achieved by using the library NGX Bootstrap and adding a btnCheckbox attribute to each button within a group.
