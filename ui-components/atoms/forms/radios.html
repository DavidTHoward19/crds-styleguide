Radios

One (and only one) option must be selected. No option is pre-selected by default. Once an option is touched, a user cannot return to the unselected state of the component.

The button groups featured in this section have radio button behaviour associated with them. This is achieved by using the library NGX Bootstrap and adding [(ngModel)]="radioModel" btnRadio="*" to each button within a group.
