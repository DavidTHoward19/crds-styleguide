Tooltips

Tooltips must be initialized with jQuery: select the specified element and call the tooltip() method.

The following code will enable all tooltips on the document:

{% highlight js %} $(document).ready(function(){ $('[data-toggle="tooltip"]').tooltip(); });{% endhighlight %}
