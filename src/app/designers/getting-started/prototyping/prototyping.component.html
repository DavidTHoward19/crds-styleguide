Step 3: Download the DDK

  1. Go to the GH repo (https://github.com/crdschurch/crds-design)
  2. Click the green button (Clone or download) and select Download Zip
  3. Unzip the folder

Step 4: Load the Library into Sketch

  1. Hang tight, there are just a few more hoops to jump through.
  2. In Sketch, start a new document or open an existing one
  3. If the Craft panel is not visible, choose Craft > Toggle Panel
  4. Click the Library icon in the panel
  5. Select Import Library
  6. Open the unzipped folder (the one you downloaded from Github)
