Content Blocks

When integrating content blocks from Crossroads' CMS within your Angular 2 application, you can use the crds-ng2-content-block module.

Install the NPM package like so...

npm install crds-ng2-content-block --save

Update your app module by importing & adding the ContentBlockModule to your imports array...

The argument passed to forRoot() refers to the category associated with the content block in the CMS.

You only need to call forRoot() once; any child components of your application that need to invoke the component can just import ContentBlockModule which will prevent multiple XHR requests to the CMS.


Once you've imported the module, you can use the component in HTML, like so...

<crds-content-block id="ContentBlockTitle"></crds-content-block>

For more details, checkout the project's Github repository.
