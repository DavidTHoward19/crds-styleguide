Developers, Start Your IDEs

Proper implementation of brand-standards is hard enough — even in traditional media. In a digital context, adhering to well-intentioned guidelines is not only tricky but can be difficult to maintain over the lifetime of any project. DDK to the rescue!

This website was specifically developed to empower developers with all the implementation details they need to be successful while working on digital properties for Crossroads. This approach allows the brand to maintain cohesion & consistency through the use of well-established patterns by helping deliver quality user-experience with minimal effort.

One Package to Rule Them All

By building your work on-top of the patterns described within the DDK, you’ll be leaning on approved, established styles.

No more guess-work around implementation; no more wondering if your CSS will get rejected during acceptance review; simply copy the provided example code, override where needed and move on with your day.

See the NPM Package

Built On Bootstrap

We’ve taken the Bootstrap CSS Framework you know and love and extended it to support approved patterns that are in use all across Crossroads’ digital properties. Through the use of NPM and other related technologies, simply include one package and you’re off to the races.
