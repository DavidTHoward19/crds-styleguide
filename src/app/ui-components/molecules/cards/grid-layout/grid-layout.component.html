Card Image Heights

Card images are set to 200px tall, cropped, and centered by default. This normalizes card sizes without need for the user to maintain consistent dimensions with their uploads.

You can override this behavior by adding a .card-img-unrestrained selector to the image(s). The example below shows the effect of these selectors.
