Default Style Modals

The following modal examples use the ngx-bootstrap library to illustrate expected styles and behaviour in an Angular 2+ project. Each examples' html tab is applicable to non-Angular projects using Bootstrap and jQuery as a dependency.

There are three distinct styles of modal available in the DDK: a default modal, a video modal, and a full-screen modal. Implementing any desired type is easy - just add a modifying class to the modal's wrapping <div> (this contains a class of .modal).

Full-Screen Modals

Full-screen modals slightly deviate from this pattern in that they require an additional sizing class modifier to achieve the full-screen effect. For this modal to work, you must add a class of .modal-full to the .modal-dialog <div>. Full-screen modals also use the alpha class in place of the fade class to achieve a smoother animation.
