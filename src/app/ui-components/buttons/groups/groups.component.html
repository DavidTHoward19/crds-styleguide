For a button to have an active state, the class active must be added to the selector. This can be done using either class binding, .e.g. [class.active]="isActive", or with the NgClass directive, e.g. [ngClass]="{'active'}".

For demonstration purposes, the active class has been applied to the first button within each button group to show what active state buttons will look like.
