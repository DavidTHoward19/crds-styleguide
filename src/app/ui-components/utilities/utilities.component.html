UI Components / Utility Classes

Utility Classes

Corners

To give an element rounded corners, use the class .round-corners. If you want to remove rounded corners from an element, use a class of .square-corners.

Please Note: .round-corners relies on a Bootstrap-defined variable. See documentation for that variable here.

Group of Classes CSS border-radius
.round-corners border-radius: $border-radius-base;
.square-corners border-radius: 0;

Adding a class of .push-* to an element will increase its desired margin.

margin-left: $line-height-computed;
.push-half margin: $line-height-computed / 2;
.push-half-top margin-top: $line-height-computed / 2;

.push-half-left margin-left: $line-height-computed / 2;

margin-left: $line-height-computed / 2;
.push-quarter-bottom margin-bottom: $line-height-computed / 4;

.flush-left margin-left: 0;

margin-left: 0;

<div class="vertical-bottom">...</div>

<div class="vertical-20">...</div>

<div class="vertical-80">...</div>

  top: 50%;
