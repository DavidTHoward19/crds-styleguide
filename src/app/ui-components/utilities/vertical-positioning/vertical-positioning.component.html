Vertical Positioning

To use these utility classes, add to an element whose parent has position: relative.

All vertical positioning utility classes have position: absolute and width: 100%.

Group of Classes CSS
.vertical-center left: 50%;
top: 50%;
.vertical-bottom left: 50%;
top: 0;
.vertical-10 left: 50%;
top: 10%;
.vertical-20 left: 50%;
top: 20%;

top: 70%;
